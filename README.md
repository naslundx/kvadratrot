# kvadratrot

Webbplats för boken "Verklighetens Kvadratrötter".
Originaltema baserat på INFINITY från styleshout.

## Utveckling

Projektet använder ESLint för JavaScript och Stylelint för CSS.

### Förutsättningar

- Node.js installerat.
- Kör `npm install` för att installera beroenden.

### Linter

För att köra linter-kontroller för både JS och CSS:

```sh
npm run lint
```

För att köra specifika kontroller:

```sh
npm run lint:js
npm run lint:css
```
